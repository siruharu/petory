# Task 01

## Goal
`app/`를 실행 가능한 Expo 프로젝트로 인식시키기 위한 최소 메타 파일을 추가한다.

## Files
- app/package.json
- app/tsconfig.json
- app/babel.config.js
- app/app.json

## Requirements
- Expo 기준의 최소 의존성 선언을 추가한다.
- TypeScript 설정을 추가한다.
- Babel 설정 파일을 추가한다.
- Expo 앱 메타데이터 파일을 추가한다.

## Constraints
- 선행 작업 없음.
- 실행 베이스만 추가하고 앱 로직은 구현하지 않는다.
- Expo 선택은 이미 확정된 결정으로 간주한다.

## Done Criteria
- `app/`가 패키지 디렉터리로 인식된다.
- Expo 실행과 런타임 진입 파일 추가를 위한 기반이 생긴다.
