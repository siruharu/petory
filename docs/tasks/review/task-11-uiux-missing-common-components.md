# Task 11: UIUX Missing Common Components

## Goal

`plan-uiux.md`에 정의했지만 실제로 빠진 공통 UI 컴포넌트를 보강한다.

## Files

- `app/src/components/common/`
- `app/src/components/feedback/`
- `app/src/components/cards/`
- `app/src/components/forms/`

## Requirements

- 우선순위 높은 누락 컴포넌트 선별
- `section-header`, `status-chip`, `form-section` 또는 동등 역할 컴포넌트 보강
- 화면별 ad-hoc style 중복을 줄일 수 있는 최소 공통 레이어 추가

## Constraints

- 디자인 방향을 다시 바꾸지 않는다
- 이미 만든 theme 토큰은 유지한다

## Done Criteria

- 계획 대비 핵심 공통 UI 누락분이 줄어든다
