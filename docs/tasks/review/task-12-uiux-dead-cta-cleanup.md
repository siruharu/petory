# Task 12: UIUX Dead CTA Cleanup

## Goal

보이지만 동작하지 않는 주요 CTA를 제거하거나 실제 동작과 연결한다.

## Files

- `app/src/screens/pets/pet-switcher-sheet.tsx`
- 관련 화면 파일

## Requirements

- `새 반려동물 등록` 같은 dead CTA 처리
- 실제 연결이 없다면 hidden 또는 disabled 처리
- CTA 진실성 원칙을 화면에 반영

## Constraints

- navigation 구조 전체 도입은 하지 않는다
- 새로운 기능을 추가하지 않는다

## Done Criteria

- 사용자에게 가능한 행동처럼 보이는 dead CTA가 남지 않는다
