# Task 05: Dashboard Empty State First Pet

## Goal

반려동물이 없는 상태에서 홈 대시보드가 첫 행동 시작점 역할을 하도록 빈 상태 UX를 보강한다.

## Files

- `app/src/screens/home/home-screen.tsx`

## Requirements

- 반려동물 0마리 상태에서 첫 등록 CTA가 명확히 보이도록 정리
- “정보가 없어요” 수준에서 끝나지 않고 다음 행동이 드러나야 함
- 홈 진입 직후 사용자가 무엇을 해야 하는지 알 수 있게 문구/버튼 배치 조정

## Constraints

- 홈 레이아웃을 전면 재설계하지 않는다
- 일정/기록 섹션 구조는 유지한다

## Done Criteria

- 반려동물이 없을 때 홈 화면에서 첫 등록 행동이 바로 가능하다
