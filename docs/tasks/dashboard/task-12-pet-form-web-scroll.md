# Task 12: Pet Form Web Scroll

## Goal

웹 프리뷰에서 반려동물 등록 화면 하단까지 항상 접근 가능하도록 스크롤 레이아웃을 정리한다.

## Files

- `app/src/components/common/screen-container.tsx`
- `app/src/screens/pets/pet-form-screen.tsx`
- 필요 시 `app/src/components/forms/pet-form.tsx`

## Requirements

- 웹에서 화면 높이가 작아도 하단 폼 영역과 제출 버튼까지 접근 가능
- 스크롤 가능한 컨테이너 구조 반영
- 모바일과 웹 모두에서 레이아웃이 크게 깨지지 않도록 유지

## Constraints

- 전체 앱 레이아웃 시스템을 다시 설계하지 않는다
- 반려동물 등록 화면 문제 해결 범위로 제한한다

## Done Criteria

- 웹에서 창 높이가 작아도 반려동물 등록 폼 하단까지 스크롤 가능하다
