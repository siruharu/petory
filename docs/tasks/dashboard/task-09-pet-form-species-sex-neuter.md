# Task 09: Pet Form Species Sex Neuter

## Goal

반려동물 등록 폼의 종, 성별, 중성화 상태를 선택형 입력으로 전환한다.

## Files

- `app/src/components/forms/pet-form.tsx`
- `app/src/screens/pets/pet-form-screen.tsx`

## Requirements

- `species`를 드롭다운 선택으로 전환
- `sex`를 드롭다운 선택으로 전환
- `neuteredStatus`를 드롭다운 선택으로 전환
- 기존 제출 payload 구조와 호환 유지

## Constraints

- 서버 API 계약은 바꾸지 않는다
- 반려동물 폼 전체 레이아웃을 전면 재설계하지 않는다

## Done Criteria

- 종, 성별, 중성화 상태가 자유 텍스트가 아니라 명시적 선택 UX로 동작한다
