# Task 07: Pet Form Option Datasets

## Goal

반려동물 등록 폼에서 사용할 선택형 입력 기준값을 정의한다.

## Files

- `app/src/components/forms/`
- 필요 시 `app/src/types/`

## Requirements

- `species` 옵션 정의
- `sex` 옵션 정의
- `neuteredStatus` 옵션 정의
- `dog`, `cat` 품종 후보군 초안 정의
- 폼과 화면에서 재사용 가능한 상수 구조로 정리

## Constraints

- 외부 API 연동으로 품종 데이터를 가져오지 않는다
- MVP 범위를 넘는 대규모 품종 데이터베이스는 만들지 않는다

## Done Criteria

- 반려동물 폼이 선택형 입력에 필요한 값 집합을 참조할 수 있다
