# Task 13: Pet Form Review

## Goal

반려동물 등록 폼 UX 개선 이후 실제로 입력 경험과 웹 레이아웃 문제가 해소됐는지 검토하고 리뷰 문서를 갱신한다.

## Files

- `docs/review-dashboard.md`
- 필요 시 `README.md`

## Requirements

- 종/품종/성별/중성화 입력 UX 점검
- 생일 선택과 나이 자동 계산 점검
- 웹 스크롤 접근성 점검
- 해결된 항목과 남은 리스크를 리뷰 문서에 반영

## Constraints

- 새로운 기능을 추가하지 않는다
- 검증 결과에 따른 최소 문서 수정만 허용한다

## Done Criteria

- `review-dashboard.md`가 최신 반려동물 등록 UX 기준으로 갱신된다
- 남은 문제와 해결된 항목이 명확히 분리된다
