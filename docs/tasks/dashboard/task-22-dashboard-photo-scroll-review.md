# Task 22: Dashboard Photo And Scroll Review

## Goal

이미지 등록과 홈 웹 스크롤 수정 이후 대시보드/반려동물 등록 흐름을 다시 검토하고 리뷰 문서를 갱신한다.

## Files

- `docs/review-dashboard.md`

## Requirements

- 반려동물 이미지 입력/preview 상태를 현재 기준으로 기록한다
- 홈 웹 스크롤이 해소됐는지 문서에 반영한다
- 새로 생긴 남은 리스크가 있으면 추가한다

## Constraints

- 기능 구현은 포함하지 않는다
- 검토 결과에 따른 문서 수정만 수행한다

## Done Criteria

- `review-dashboard.md`가 최신 상태로 갱신된다
- 이미지 등록과 홈 웹 스크롤 관련 해결 여부가 문서에 명시된다
