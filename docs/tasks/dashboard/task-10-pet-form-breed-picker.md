# Task 10: Pet Form Breed Picker

## Goal

종 선택에 따라 품종 후보군이 달라지는 입력 UX를 반려동물 등록 폼에 추가한다.

## Files

- `app/src/components/forms/pet-form.tsx`
- 필요 시 `app/src/components/forms/`

## Requirements

- `species` 값에 따라 `breed` 옵션 목록이 변경되도록 연결
- MVP 기준으로 `dog`, `cat` 품종 후보군 제공
- 드롭다운 또는 단순 검색형 선택 방식 중 현재 구조에 맞는 쪽으로 구현

## Constraints

- 고급 자동완성 라이브러리 도입은 하지 않는다
- 모든 품종을 exhaustive하게 다루려 하지 않는다

## Done Criteria

- 사용자가 개/고양이 품종을 후보군에서 선택하거나 찾을 수 있다
