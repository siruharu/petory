# Task 06: Dashboard Flow Review

## Goal

로그인 후 대시보드에서 반려동물 등록/전환/관리 흐름이 실제로 닫혔는지 검증하고 리뷰 문서를 갱신한다.

## Files

- `docs/review-dashboard.md`
- 필요 시 `README.md`

## Requirements

- 반려동물 0마리 상태 점검
- 반려동물 1마리 이상 상태 점검
- 홈 -> 반려동물 등록/목록 진입 가능 여부 점검
- 등록 후 복귀 흐름 점검
- 해결된 항목과 남은 리스크를 문서에 반영

## Constraints

- 새로운 기능을 추가하지 않는다
- 검증 결과에 따른 최소 문서 수정만 허용한다

## Done Criteria

- `review-dashboard.md`가 최신 구현 상태 기준으로 갱신된다
- 대시보드 행동 경로가 닫혔는지 문서로 확인 가능하다
