# Task 08: Form Select Component

## Goal

반려동물 등록 폼에서 사용할 드롭다운 또는 선택형 입력 컴포넌트를 추가한다.

## Files

- `app/src/components/forms/`

## Requirements

- 기존 `text-field`와 함께 사용할 수 있는 선택형 입력 컴포넌트 추가
- 라벨, 현재 값, 옵션 목록, 선택 이벤트를 수용하는 최소 구조 제공
- 모바일/웹 모두에서 동작 가능한 단순 구현 우선

## Constraints

- 복잡한 커스텀 bottom sheet 선택 UI까지 확장하지 않는다
- 외부 UI 라이브러리를 새로 도입하지 않는다

## Done Criteria

- 반려동물 폼이 자유 텍스트 대신 선택형 입력 컴포넌트를 사용할 수 있다
