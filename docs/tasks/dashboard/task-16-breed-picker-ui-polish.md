# Task 16: Breed Picker UI Polish

## Goal

품종 검색/선택 UI가 웹에서 깨져 보이지 않도록 시각 구조와 상호작용을 정리한다.

## Files

- `app/src/components/forms/search-select-field.tsx`
- `app/src/components/forms/pet-form.tsx`

## Requirements

- 현재 선택값, 검색 input, 옵션 목록이 시각적으로 분리되어 보여야 한다
- 옵션 리스트가 너무 넓게 퍼지거나 무너져 보이지 않아야 한다
- 웹 프리뷰 기준으로 선택 가능한 리스트라는 인지가 쉬워야 한다

## Constraints

- 기존 품종 데이터셋 구조는 유지한다
- 새로운 디자인 시스템이나 외부 라이브러리를 추가하지 않는다
- 검색형 선택 UX는 유지한다

## Done Criteria

- 품종 선택 UI가 웹에서 읽기 쉽고 정돈된 형태로 보인다
- 검색과 선택이 기존보다 명확하게 구분된다
- 기존 species 연동 동작이 유지된다
