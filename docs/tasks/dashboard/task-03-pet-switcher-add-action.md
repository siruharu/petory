# Task 03: Pet Switcher Add Action

## Goal

대시보드의 반려동물 전환 섹션에서 새 반려동물 등록 행동을 실제로 수행할 수 있게 만든다.

## Files

- `app/src/screens/home/home-screen.tsx`
- `app/src/screens/pets/pet-switcher-sheet.tsx`

## Requirements

- 홈에서 `PetSwitcherSheet`에 `onAddPet` 전달
- 반려동물 0마리와 1마리 이상 상태 모두에서 등록 행동이 노출되도록 정리
- 기존 전환 기능은 유지

## Constraints

- PetSwitcherSheet의 시각 방향을 다시 설계하지 않는다
- 반려동물 선택 로직 자체는 바꾸지 않는다

## Done Criteria

- 대시보드의 전환 섹션에서 새 반려동물 등록으로 이동할 수 있다
