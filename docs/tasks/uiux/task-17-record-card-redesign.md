# Task 17: Record Card Redesign

## Goal

기록 카드를 타임라인형 정보 카드로 재설계한다.

## Files

- `app/src/components/cards/record-card.tsx`

## Requirements

- 기록 타입, 시각, 핵심 값, 메모 위계 정리

## Constraints

- 새로운 record type은 추가하지 않는다
- 수정/삭제 액션은 추가하지 않는다

## Done Criteria
