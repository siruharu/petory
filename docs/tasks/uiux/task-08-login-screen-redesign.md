# Task 08: Login Screen Redesign

## Goal

로그인 화면을 2026년형 입력 집중 구조로 재구성한다.

## Files

- `app/src/screens/auth/login-screen.tsx`

## Requirements

- 상단 intro 영역 추가
- text field 공통 컴포넌트 사용
- submit error와 verification 안내를 시각적으로 분리
- primary CTA와 secondary action 위계 정리

## Constraints

- 로그인 API 로직은 바꾸지 않는다
- 비밀번호 찾기 흐름은 추가하지 않는다

## Done Criteria

- 로그인 화면이 입력, 오류, 보조 액션의 계층을 명확히 가진다
