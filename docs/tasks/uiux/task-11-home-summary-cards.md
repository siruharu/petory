홈 화면에 사용할 요약 카드 컴포넌트를 추가한다.

## Files

- `app/src/components/cards/dashboard-summary-card.tsx`

- 선택된 반려동물 카드

- 홈 화면에서 재사용 가능한 핵심 카드 컴포넌트가 준비된다
