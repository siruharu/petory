# Task 12: Home Screen Layout

## Goal

홈 화면을 요약 중심 대시보드 구조로 재배치한다.

## Files

- `app/src/screens/home/home-screen.tsx`

## Requirements

- 상단 pet context hero 구성
- today summary, overdue, recent records, quick actions 섹션 정리
- 개발용 `Refresh Home` 버튼 제거 또는 보조 액션 수준으로 축소
- loading/empty/error를 공통 feedback 컴포넌트로 교체

## Constraints

- API 호출 구조는 유지한다
- 새로운 홈 데이터 필드는 추가하지 않는다

## Done Criteria

- 홈 화면이 “오늘의 돌봄 대시보드”처럼 보이는 레이아웃을 갖는다
