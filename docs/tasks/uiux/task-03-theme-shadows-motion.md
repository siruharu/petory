# Task 03: Theme Shadows Motion

## Goal

shadow와 motion 토큰을 정의해 공통 테마 레이어를 완성한다.

## Files

- `app/src/theme/shadows.ts`
- `app/src/theme/motion.ts`
- `app/src/theme/index.ts`

## Requirements

- 얕은 카드 shadow 단계 정의
- transition duration/easing 토큰 정의
- 이후 카드/시트 컴포넌트에서 재사용 가능한 형태로 export

## Constraints

- 실제 animation 구현은 하지 않는다
- 플랫폼별 분기 로직은 넣지 않는다

## Done Criteria

- shadow와 motion 토큰 파일이 추가된다
- index에서 재export된다
