# Task 19: Form Shells Redesign

## Goal

반려동물/일정/기록 폼을 공통 form shell 기준으로 재정리한다.

## Files

- `app/src/components/forms/pet-form.tsx`
- `app/src/components/forms/schedule-form.tsx`
- `app/src/components/forms/record-form.tsx`

## Requirements

- 공통 field group 컴포넌트 사용
- 입력 섹션을 시각적으로 그룹화
- submit 버튼과 form body의 위계 정리

## Constraints

- 폼 저장 로직은 바꾸지 않는다
- 필드 종류 추가는 하지 않는다

## Done Criteria

- 세 폼 컴포넌트가 공통 시각 규칙으로 읽힌다
