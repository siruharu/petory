# Task 05: Feedback State Components

## Goal

loading, empty, error, inline message 컴포넌트를 공통화한다.

## Files

- `app/src/components/feedback/loading-state.tsx`
- `app/src/components/feedback/empty-state.tsx`
- `app/src/components/feedback/error-state.tsx`
- `app/src/components/feedback/inline-message.tsx`

## Requirements

- 제목, 설명, CTA를 받을 수 있는 empty/error 컴포넌트
- 간단한 block placeholder 또는 텍스트 기반 loading 컴포넌트
- success/error/info tone을 받을 수 있는 inline message

## Constraints

- 아이콘 라이브러리 추가 금지
- skeleton shimmer 애니메이션은 넣지 않는다

## Done Criteria

- 상태 UI가 문자열 반복 없이 공통 컴포넌트로 재사용 가능하다
