# Task 04: Screen Container Shell

## Goal

기존 `ScreenContainer`를 UI 토큰 기준의 공통 화면 셸로 정리한다.

## Files

- `app/src/components/common/screen-container.tsx`

## Requirements

- 공통 배경색 적용
- 기본 horizontal padding 정리
- top/bottom spacing 일관화
- children을 감싸는 기본 column layout 제공

## Constraints

- 스크롤 구조까지 추가하지 않는다
- 각 화면 전용 헤더 기능은 넣지 않는다

## Done Criteria

- `ScreenContainer`가 테마 토큰을 사용한다
- 주요 화면에서 공통 여백 기준으로 재사용 가능한 상태가 된다
