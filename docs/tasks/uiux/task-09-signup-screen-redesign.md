# Task 09: Signup Screen Redesign

## Goal

회원가입 화면을 가입 입력 상태와 인증 대기 상태로 분리해 재구성한다.

## Files

- `app/src/screens/auth/signup-screen.tsx`

## Requirements

- 입력 상태 레이아웃 정리
- 가입 성공 후 verification pending success panel 추가
- 메일 재전송 버튼을 보조 CTA로 배치

## Constraints

- 실제 signup/resend 로직은 유지한다
- 약관/마케팅 수신 등 추가 필드는 넣지 않는다

## Done Criteria

- 회원가입 성공 후 사용자가 다음 행동을 명확히 이해할 수 있다
