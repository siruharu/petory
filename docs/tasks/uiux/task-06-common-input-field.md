# Task 06: Common Input Field

## Goal

로그인/회원가입/폼 화면에서 공통으로 사용할 입력 컴포넌트를 만든다.

## Files

- `app/src/components/forms/text-field.tsx`
- `app/src/components/forms/field-group.tsx`

## Requirements

- label, placeholder, error message, helper text를 표현할 수 있는 입력 컴포넌트
- spacing과 border 상태를 공통 처리
- secureTextEntry 등 기본 TextInput props 전달 가능

## Constraints

- 입력 validation 로직은 넣지 않는다
- 멀티라인 전용 컴포넌트까지 확장하지 않는다

## Done Criteria

- 인증 및 CRUD 폼에서 재사용 가능한 기본 입력 컴포넌트가 생긴다
