# Task 01: API Contract 초안 작성

## Goal
`docs/api-contract.md`를 생성해서 모바일과 백엔드가 공통으로 따를 인증, 반려동물, 일정, 기록, 홈, 알림 API 계약 초안을 정의한다.

## Files
- docs/api-contract.md

## Requirements
- 인증 API 요청/응답 형식을 정의한다.
- 반려동물 목록/생성/수정 API 형식을 정의한다.
- 일정 목록/생성/완료 API 형식을 정의한다.
- 기록 목록/생성 API 형식을 정의한다.
- 홈 집계 API 응답 형식을 정의한다.
- 에러 응답 공통 구조를 정의한다.

## Constraints
- 선행 작업 없음.
- 구현 코드는 작성하지 않는다.
- `docs/plan.md`와 현재 스켈레톤 파일 구조를 기준으로 작성한다.

## Done Criteria
- `docs/api-contract.md`가 생성되어 있다.
- 각 API의 method, path, request, response 예시가 있다.
- 모바일/백엔드가 동일한 응답 필드명을 기준으로 개발할 수 있다.
