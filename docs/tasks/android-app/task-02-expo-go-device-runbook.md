# Task 02: Expo Go Device Runbook

## Goal

현재 코드베이스를 안드로이드 폰의 Expo Go에서 실제로 띄우고 로그인/홈까지 검증하는 실행 절차를 문서로 정리한다.

## Files

- 관련 구현/가이드 문서
- 필요 시 `docs/implementation/` 하위 Android 실행 문서

## Requirements

- Expo Go 실행 순서가 단계별로 정리돼야 한다
- 서버 실행, LAN 연결, QR 접속, 로그인, 홈 확인 순서가 포함돼야 한다
- 폰과 개발 머신이 같은 네트워크에 있어야 한다는 전제가 명시돼야 한다
- 실패 시 확인할 포인트가 포함돼야 한다

## Constraints

- 앱 구조를 새로 만들지 않는다
- EAS Build나 APK 배포 단계까지 포함하지 않는다
- 실행 문서는 현재 코드 기준으로만 작성한다

## Done Criteria

- 새 개발자도 문서만 보고 Expo Go 실기기 테스트를 재현할 수 있다
- 로그인/세션 복원/홈 조회까지의 검증 체크리스트가 문서에 남는다
